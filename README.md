# Movie Recommendation System

## Overview
A Python-based GUI that recommends top 10 movies based on user-selected genre using IMDb dataset.

## Features
- Filter movies by genre
- Display top-rated movies using a simple GUI (Tkinter)
- Handles invalid input and empty results gracefully

## Tech Stack
- Python (pandas, Tkinter)
- Dataset: IMDB-Movie-Data.csv

## How to Run
1. Download the repo
2. Make sure `IMDB-Movie-Data.csv` is in the same folder
3. Run:
```bash
python movie_recommender.py
